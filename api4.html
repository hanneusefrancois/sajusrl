📡 API Reference — All Data Sources

Frankfurter (ECB) FREE

30+ currencies. Historical from 1999. No key. CORS enabled.

GET https://api.frankfurter.app/latest?base=EUR 🔗 Documentation

World Bank FREE

16,000+ indicators. 200+ countries. GDP, inflation, trade.

GET https://api.worldbank.org/v2/country/WLD/indicator/FP.CPI.TOTL.ZG?format=json 🔗 Documentation

Eurostat FREE

EU-27 PPI, labour costs, energy, trade. JSON-stat format.

GET https://ec.europa.eu/eurostat/api/dissemination/statistics/1.0/data/sts_inpr_m 🔗 Documentation

FRED (St. Louis Fed) KEY

800k+ series: CPI, PPI, gold, oil, rates. Free registration.

GET https://api.stlouisfed.org/fred/series/observations?series_id=GOLDAMGBD228NLBM 🔑 Get Free Key

Open Exchange Rates KEY

170+ currencies. Real-time + historical. 1,000 req/month free.

GET https://openexchangerates.org/api/latest.json?app_id=YOUR_KEY 🔗 Documentation

Metals.dev KEY

Gold, silver, platinum, copper, aluminum. Spot prices 20+ currencies.

GET https://api.metals.dev/v1/latest?api_key=YOUR_KEY&currency=EUR 🔗 Documentation

BIS Statistics FREE

Central bank data: effective exchange rates, credit, property prices. Since 1930.

GET https://stats.bis.org/api/v2/data/WS_EER/... 🔗 Documentation
