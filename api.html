🔌 Free API directory · No key required

Open data APIs for price modelling.

All APIs below are free, require no authentication, and support CORS for browser-side fetching.

frankfurter.app

✅ Free · No key · CORS · ECB data

Exchange rates for 30+ currencies. Historical data from 1999. Based on ECB reference rates.

GET https://api.frankfurter.app/latest?base=EUR
🔗 Documentation

World Bank API

✅ Free · No key · 16,000+ indicators

GDP, inflation, commodity prices, trade data for 200+ countries. The backbone of macro calibration.

GET api.worldbank.org/v2/country/WLD/indicator/FP.CPI.TOTL.ZG?format=json
🔗 Documentation

FRED API (St. Louis Fed)

✅ Free key · 800,000+ series

The world's largest economic database. Commodity prices, CPI, PPI, interest rates, employment data.

GET api.stlouisfed.org/fred/series/observations?series_id=GOLDAMGBD228NLBM
🔗 Documentation

Open Exchange Rates

✅ Free tier · 170+ currencies

Real-time and historical FX rates. Free plan: 1,000 requests/month. Ideal for currency risk modelling.

GET openexchangerates.org/api/latest.json
🔗 Documentation

Metals.dev

✅ Free tier · Precious + industrial metals

Gold, silver, platinum, palladium, copper, aluminum. Real-time spot prices in 20+ currencies.

GET api.metals.dev/v1/latest?api_key=YOUR_KEY&¤cy=EUR
🔗 Documentation

Eurostat API

✅ Free · No key · EU harmonised data

Industrial PPI, labour costs, energy prices, trade statistics. JSON/SDMX format. EU-27 harmonised.

GET ec.europa.eu/eurostat/api/dissemination/sdmx/2.1/data/STS_INPR_M
